What's kernelci.org?
kernelci.org, this website, is a build and boot automation tool for upstream Linux kernel trees.
Where can I find the code that kernelci.org is built on?
The code powering kernelci.org can be found at the Linaro Git Code Hosting: https://git.linaro.org.
There are two main components:
  1. This website
  2. The Kernel CI API
What's a job?
A job is a collection of build and boot data for a given Linux kernel tree/branch.
How can I participate in the build test phase?
At the moment we do not have plans for other parties to participate in the build test phase.
How can I participate in the boot test phase?
At the moment we do not have plans for other parties to participate in the boot test phase.
I would like to have my board boot tested, how can I do it?
At the moment we only support some specific boards.
Can I send you my boards?
At the moment we do not have plans to receive or accept new boards. For more info, please contact us.
